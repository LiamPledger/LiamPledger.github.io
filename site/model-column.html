ML Model

ML model for estimating the drift capacity of RC columns

This page allows users to apply a machine learning model developed for estimating the drift capacity of RC columns based on given structural and geometric properties. Existing data of cyclic tests of RC columns were used to train a Gradient-Boosted Decision Tree (GBDT) machine learning algorithm to estimate drift capacity. Drift capacity was defined as the drift measured at a 20% loss in lateral resistance. A total of 341 RC column tests from the ACI-369 database are used to train the ML algorithm. The database of columns used in this study can be found in the GitHub repository.

Please get in touch if you would like to share more tests not currently included in the repository.
